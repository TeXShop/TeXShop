Getting and Installing teTeX

Gerben Wierda has compiled the latest teTeX for MacOSX. His version also includes updated programs from the TeX User's Group's distribution TeXLive. To obtain these files and programs, go to Wierda's site

http://www.rna.nl/tex.html

and obtain i-Installer, the TeX installer. You'll find it in the downloadable package II2.dmg. Run the installer and install the following packages:

  • TeX
  • Ghostscript 8
  • Freetype 2
  • wmf and iconv conversion support
  • ImageMagick
  • CM Super for TeX
  • TeX Support: TeX4ht

The first package are essential for TeX. The second package is necessary to use the TeX + Ghostscript option of TeXShop described below. The third, fourth, and fifth packages are used to convert graphic formats into formats which TeX understands. The sixth package contains an optional set of additional fonts. The final package is necessary to convert TeX source code to web pages.

If you are running Mac OS X 10.3 (Panther) or higher, you may not need to install Ghostscript. Panther contains its own distiller to convert postscript files to pdf format, and TeXShop has a preference setting to use that distiller rather than ghostscript. But you still may wish to install ghostscript, since a small number of tex packages use it directly. The most important is epstopdf.sty, which can be used to automatically convert eps files to pdf format during typesetting.

More detailed installation instructions can be found on Wierda's site, and on the TeXShop web site listed later.
