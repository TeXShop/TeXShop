Einleitung

Bitte lesen Die nur den ersten Abschnitt dieses Dokuments, wenn Sie als neuer Benutzer TeXShop zum ersten Mal installieren. Wenn Sie TeXShop von einer älteren auf diese Version aktualisiert haben, dann lesen Sie diese und die nächsten drei Abschnitte. Wenn Sie sich mit TeXShop gut auskennen und es anpassen wollen, dann lesen Sie auch die übrigen Abschnitte.

Wenn TeXShop zum ersten Mal gestartet wird, so legt es Standardeinstellungen und Standardvorlagen im „Library“-Ordner des Benutzers an. Wird das Programm später aktualisiert, so werden diese Dateien nicht überschrieben, weil der Benutzer sie eventuell an seine Bedürfnisse angepasst hat. In seltenen Fällen werden an diesen Standard-Dateien Veränderungen vorgenommen. Dies Veränderungen werden hier aufgeführt, sodass Benutzer der voherigen Version ihre angepassten Standard-Dateien entsprechend verändern können können.

TeXShop verfügt über versteckte Einstellungen für erfahrene Benutzer. Hierdurch kann das Programmverhalten durch die Installation verschiedener .plist-Dateien beeinflusst werden. Details hierüber erfahren Sie in den entsprechenden Abschnitten.

Neue Versionen von TeXShop beinhalten manchmal neue Menüeinträge oder andere neue Komponenten der Benutzeroberfläche. Diese neuen Komponenten sind manchmal noch nicht lokalisiert auch wenn der Rest des Programms auch in Ihrer Sprache vorliegt. Schicken Sie mir die passende Übersetzung, so wird diese sofort in die TeXShop Version auf meiner Webseite eingepflegt. Schreiben Sie einfach an koch@math.uoregon.edu.
