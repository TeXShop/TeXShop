Autovervollständigen

Greg Landweber hat die Funktion „Autovervollständigen“ TeXShop hinzugefügt. Wenn Sie Anführungszeichen tippen, wird automatisch ein zweites Paar erstellt und der Cursor dazwischen positioniert. Tippen Sie ein ^ ergibt das ^{ } mit dem Cursor zwischen den geschweiften Klammern. 29 solcher Autovervollständigungen sind möglich.

Autovervollständigen kann in den „Einstellungen...“ abgeschaltet werden. Die Standardeinstellung ist Aus. Um Autovervollständigen auszuprobieren müssen Sie die Funtkion erstmal aktivieren.

Autovervollständigen ist konfigurierbar. Um dies zu tun öffnen Sie die Datei ~/Library/TeXShop/Keyboard/completion.plist mit TeXShop. Lesen Sie die Anmerkung am Anfang des Dokumentes, passen Sie Landwebers Auswahl an oder fügen Sie Ihre eigene hinzu. Speichern Sie die Datei (im UTF-8 Format falls Sie Unicode benutzen) ab.
