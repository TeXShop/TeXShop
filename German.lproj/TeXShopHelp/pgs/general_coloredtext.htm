Farbiger Text in TeX

Es war schon immer möglich auch farbige TeX-Dokumente auszugeben. Fügen Sie einfach folgende Zeile zur Präambel Ihres Quelltextes hinzu:

\usepackage{color}

Fügen Sie jetzt zum Beispiel folgende Zeile vor dem Setzen ein:

\color{blue}

Um nur einen bestimmten Teil des Dokuments farbig zu gestalten fügen Sie folgendes ein:

{\color{red} Das ist sehr wichtig!}

Um spezielle Farben zu erzeugen, benutzen Sie Befehle wie:

\definecolor{mycolor}{rgb}{0.2, 0.7, 0.8}

Farbiger Text kann auf dem Bildschirm schwer zu lesen sein. Daher kann man bei TeXShop durch eine versteckte Einstellung die Hintergrundfarbe des Vorschau-Fensters einstellen. Diese Hintergrundfarbe wird nicht mit ausgedruckt. Um den Hintergrund zum Besipiel grau zu färben, geben Sie im Terminal folgenden Befehl ein:

defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_R 0.5
defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_G 0.5
defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_B 0.5

Ändern Sie „0.5“ in „1“ um zu einem weißen Hintergrund zurück zu kehren. Wollen Sie den Hintergrund häufig ändern, erstellen Sie sich ein AppleScript, dass die Hintergrundfarbe auf grau setzt und den Namen „PDF Hintergrund grau“ trägt. Sie können hierzu den Makro-Editor benutzen:

--AppleScript
do shell script "defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_R 0.5"
do shell script "defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_G 0.5"
do shell script "defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_B 0.5"

Erstellen Sie ein ähnliches Skript, dass den Hintergrund wieder auf weiß setzt und nennen Sie es „PDF Hintergrund weiß“.
