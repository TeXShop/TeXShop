Einen externen Editor verwenden

TeXShop kann über einen externen Editor konfiguriert werden. Dies geschieht über den Menüpunkt „Zur Voransicht öffnen...“. Sie werden, nachdem Sie diesen Menüpunkt ausgewählt haben darum gebeten eine Quelltextdatei auszuwählen. es wird nur das Voransichts-Fenster mit der pdf-Datei geöffnet (Allerdings muss der Quelltext vorher gesetzt worden sein, sonst erscheint eine leere Seite). Dank Leuski's Werkzeugleiste kann das Vorschaufenster so konfiguriert werden, dass es eine Satzschaltfläche enthält. Sie können Ihren Quelltext mit jedem Editor bearbeiten. Wenn es an der Zeit ist diesen zu setzen, sichern Sie die Änderungen in Ihrem Editor, wechseln zum Vorschaufenster und setzen den Quelltext. Auf diese Art wird TeXShop ihre Quelldatei weder öffnen noch ändern. Es wird die Datei nur an TeX oder LaTeX weiterleiten.

Für Benutzer, die einen externen Editor meistens vorziehen, gibt es in den Einstellungen einen Punkt „Bei Programmstart externen Editor“. Wenn diese Einstellung ausgewählt ist, wird die Quelltextdatei auf die oben beschriebene Art und Weise geöffnet, wenn Sie im Menü „Öffnen“ oder "Zuletzt benutzte Dokumente" auswählen. Außerdem wird aus „Zur Voransicht öffnen...“ „Zum Editieren öffnen...“ und öffnet die Quelltextdatei mit dem integrierten Editor von TeXShop für die wenigen Ausnahmen in denen der integrierte Editor erwünscht ist. Die Befehle "Öffnen" und „Zuletzt geöffnete Dokumete...“ öffnen außerdem jpg, tiff, ps, pdf, dvi, log und andere Dateien unabhängig der eben bechriebenen Einstellungen.

AppleScript Unterstütztung für die „externer Editor Einstellung“ ist noch im beta-Stadium und wird für künftige Versionen weiterentwickelt. Dies wird es ermöglichen aus einem voll AppleScript tauglichen Editor zu setzen.
