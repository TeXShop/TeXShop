teTex beschaffen und installieren

Gerben Wierda hat teTeX für Mac OS X kompiliert. Seine Version beinhaltet außerdem aktualisierte Programme von der TeX User's Group Distribution TeXLive. Um diese Dateien zu bekommen, gehen Sie auf seine Seite

http://www.rna.nl/tex.html

und laden Sie i-Installer, das TeX Installationsprogramm. Sie werden es in dem ladbaren Paket II2.dmg finden. Starten Sie nun das Installationsprogramm. Es werden folgende Programme installiert:

  • TeX
  • Ghostscript 8
  • Freetype 2
  • wmf and iconv conversion support
  • ImageMagick
  • CM Super for TeX

Das erste Paket ist wesentlich für TeX. Das zweite Paket ist notwendig um TeX mit der Ghostscript Option von TeXShop zu nutzen. Eine Erklärung dazu finden Sie weiter unten. Die Pakete drei bis fünf werden benötigt um Grafiken für TeX zu konvertieren. Das letzte Paket beinhaltet eine zusätzliche Auswahl von Schriften.

Weitere detailierte Installationsanweisungen finden Sie auf Wierdas Webseite und auf der unten angegebenen TeXShop Webseite.
