teTex beschaffen und installieren

Gerben Wierda hat teTeX für Mac OS X kompiliert. Seine Version beinhaltet außerdem aktualisierte Programme von der TeX User's Group Distribution TeXLive. Um diese Dateien zu bekommen, gehen Sie auf seine Seite

http://www.rna.nl/tex.html

und laden i-Installer, das TeX Installationsprogramm. Sie werden es in dem ladbaren Paket II2.dmg finden. Starten Sie das Installationsprogramm und es werden folgende Programme installiert:

  • TeX Foundation
  • TeX Programme
  • Ghostscript 8
  • Freetype 2
  • wmf und iconv conversion support
  • ImageMagick
  • CM Super für TeX

Die zwei ersten Pakete sind wesentlich für TeX. Das dritte Paket ist notwendig um TeX mit der Ghostscript Option von TeXShop zu nutzen. Eine Erklärung dazu finden Sie weiter unten. Die Pakete vier bis sechs werden benötigt um Grafiken für TeX zu konvertieren. Das letzte Paket beinhaltet eine zusätzliche Auswahl von Schriften.

Weitere detailierte Installationsanweisungen finden Sie auf Wierdas Webseite und auf der unten angegebenen TeXShop Webseite.
