Syntax Farben

Wenn die farbige Darstellung der Syntax im Eingabefenster eingeschaltet ist, werden Kommentare rot gefärbt, Befehle erscheinen blau und die Symbole $, { und } sind dukelgrün gefärbt. Manche Benutzer möchten diese Farben vielleicht ändern. Jede Farbe wird durch Rot, Grün und Blau Anteile festgelegt. Diese Anteile sind tatsächliche Zahlen zwischen 0,0 und 1,0. Nehmen wir an, Sie möchten die Farbe für die Symbole $, { und } in Hellgrün ändern. Dies ist eine Farbe mit den Komponenten (r, g, b) = (0,0-1,0-0,0). Um die Farbe zu ändern, öffnen Sie ein Terminal-Fenster und geben Sie folgendes ein: (Beachten Sie, dass das Komma durch einen Punkt erzetzt werden muss.)

defaults write edu.uoregon.math.koch.TeXShop markerred 0.0
defaults write edu.uoregon.math.koch.TeXShop markergreen 1.0
defaults write edu.uoregon.math.koch.TeXShop markerblue 0.0

Die entsprechenden Bezeichnungen sind für Kommentare commentred, commentgreen und commentblue, für Befehle commandred, commandgreen und commandblue.
