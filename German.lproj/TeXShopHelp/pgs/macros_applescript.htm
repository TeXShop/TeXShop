AppleScript Makros

AppleScript Makros beginnen mit folgendem String:

--AppleScript

Hierbei wird der resultierende AppleScript Code ausgeführt, wenn das Makro gestartet wird. Nehmen wir zum Beispiel ein Makro mit dem Titel „pdf-Datei mit Acrobat Reader öffnen“ Wird dieses Makro ausgeführt, wenn die Quelltext-Datei aktiviert ist (und die pdf-Datei erstellt wurde), so startet der Adobe Acrobat Reader und die pdf-Ausgabe-Datei wird in diesem Programm angezeigt. Der dazu gehörige AppleScript Code lautet:

--AppleScript
tell application "Acrobat Reader 5.0"
activate
open POSIX file #PDFPATH#
end tell
