Dialoge erstellen

Mit Hilfe von AppleScript können Sie Dialoge erstellen und Benutzereingaben verarbeiten. Probieren Sie zum Beispiel folgendes AppleScript aus:

--AppleScript
--Wird ein Text-String zurueckgegeben, so wird er angezeigt.
-- Andere Eingaben werden ignoriert.
-- Beispiel: Dialog mit zurueckgegebenem Text
display dialog "Test-Dialog: Tippen Sie etwas ein" default answer "Hallo Welt!" buttons {"A", "B", "C"} default button "B"
return "Ihre Eingabe war:" & return & (the text returned of the result) & return & "Sie drückten folgenden Knopf: " & (the button returned of the result)
