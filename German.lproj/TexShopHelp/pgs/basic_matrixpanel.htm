Das Menü „Fenster“ enthält einen Menüpunkt „Matrix Palette...“. Mit Hilfe dieser Palette können Sie sehr leicht Matrizen erstellen. Für jeden Matrizen-Eintrag wird nur ein kleiner Raum zur Verfügung gestellt. Sie können jedoch die Einträge auch an anderer Stelle erstellen und in die Matrix kopieren. Diese wundervolle Palette ist die Arbeit von Jonas Zimmermann, den Sie unter zimmerleut@gmx.de erreichen können.

Seit TeXShop 1.35 können mit Hilfe der Matrix Palette auch Tabellen erstellt werden. Die Palette kann mit Hilfe der Datei „matrixpanel_1.plist“ konfiguriert werden, welche im Ordner „~/Library/TeXShop/MatrixPanel“ liegt. Die Struktur dieser Datei wurde mit der Version 1.35 geändert. Daher kann es sein, dass noch eine alte Date „matrixpanel.plist“ von Version 1.34 vorhanden ist.

Die Standardgröße der Matrizen, die mit Hilfe der Palette erzeugt werden kann über eine versteckte Einstellung verändert werden. Geben Sie zum Ändern der Einstellung folgenden Befehl im Terminal ein::

defaults write TeXShop matrixsize 12
