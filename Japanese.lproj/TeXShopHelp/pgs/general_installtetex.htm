teTeXの入手とインストール

Gerben Wierda は最新の teTeX を MacOSX に移植しています。彼のバージョンには、TeXLive（ TeX Users Group によるディストリビューション）が提供するアップデートされたプログラムも含まれています。これらのファイルやプログラムを手に入れるには、Wierda のサイトへ行きます──

http://www.rna.nl/tex.html

そのうえで i-Installer（ TeX のインストーラ）を取得してください。II2.dmg というダウンロード可能なパッケージに収められています。インストーラを実行し、以下のパッケージをインストールします：

  1. TeX
  2. Ghostscript 8
  3. Freetype 2
  4. wmf and iconv conversion support
  5. ImageMagick
  6. CM Super for TeX
  7. TeX Support: TeX4ht

最初のパッケージは TeX には絶対不可欠です。２番目のパッケージは後述する TeXShop の TeX ＋ Ghostscript オプションを利用するのに必要です。３番目、４番目、そして５番目のパッケージはグラフィック・フォーマットを TeX が理解できるフォーマットに換えるために使われます。６番目のパッケージは追加フォントのオプション・セットを含んでいます。最後のパッケージは TeX のソースコードをウェブページに変換するのに必要です。

もし Mac OS X 10.3（Panther）かそれ以上を使っているのであれば、Ghostscript をインストールする必要はないかもしれません。Panther にはポストスクリプト・ファイルを pdf フォーマットに変換する機能がそもそも具わっており、TeXShop は Ghostscript ではなくその機能を利用するように設定されています。しかしそれでもなお Ghostscript のインストールを望むかもしれません──少数の TeX パッケージでは直接 Ghostscript を使っていますから。最も重要なのは epstopdf.sty ですが、これはタイプセッティング時に eps ファイルを pdf フォーマットに自動的に変換するのに使われます。

補足：日本語 pTeX では、Ghostscript のインストールが推奨されます。

さらに詳しいインストールについての解説は Wierda のサイトにありますが、TeXShop のウェブサイトにも掲載しました。
