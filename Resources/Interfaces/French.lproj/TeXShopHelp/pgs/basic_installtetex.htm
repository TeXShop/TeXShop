The standard TeX distribution is called TeX Live. A new version is produced each year by the TeX Users Group (TUG) and distributed to members on a DVD. This distribution works on Mac OS X (both PowerPC and Intel), on Linux, on Windows, and on many other architectures.

The MacTeX working group of the TeX Users Group (TUG) has produced an Apple install package containing this distribution and everything else needed to run TeX on Mac OS X. The package installs in a couple of minutes with almost no user intervention. It contains a very complete version of TeX Live, TeXShop, BibDesk, the Excalibur spell checker, and Gerben Wierda's i-Installer. For up to date links to the package, see

www.tug.org/mactex/

The MacTeX package can be obtained from numerous CTAN sites. If the link on the previous site files, go to

www.ctan.org/
and click on the link "our directories" and then on the link "systems" in the resulting web page, and then on the link "mac" on the next page, and finally on the link "mactex" on the final page.
