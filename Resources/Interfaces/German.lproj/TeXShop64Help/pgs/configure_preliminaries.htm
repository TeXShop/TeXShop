Haben Sie TeXShop zum ersten Mal installiert, so lesen Sie bitte diese zwei Abschnitte sowie die darauf folgenden vier. Wenn Sie sich mit TeXShop gut auskennen und es anpassen wollen, dann lesen Sie auch die übrigen Abschnitte.

Wenn TeXShop zum ersten Mal gestartet wird, so legt es Standardeinstellungen, Makros, Standardvorlagen und Dokumentsatzprogramme im „Library“-Ordner des Benutzers an. Wird das Programm später aktualisiert, so werden diese Dateien nicht überschrieben, weil der Benutzer sie eventuell an seine Bedürfnisse angepasst hat. In seltenen Fällen werden an diesen Standard-Dateien Veränderungen vorgenommen. Diese Veränderungen werden hier aufgeführt, sodass Benutzer der voherigen Version ihre angepassten Standard-Dateien entsprechend verändern können können.

TeXShop verfügt über versteckte Einstellungen für erfahrene Benutzer. Hierdurch kann das Programmverhalten durch die Installation verschiedener .plist-Dateien beeinflusst werden. Details hierüber erfahren Sie in den entsprechenden Abschnitten.

Neue Versionen von TeXShop beinhalten hin und wieder neue Menüeinträge oder andere neue Komponenten der Benutzeroberfläche. Diese neuen Komponenten sind manchmal noch nicht lokalisiert auch wenn der Rest des Programms in Ihrer Sprache vorliegt. Schicken Sie mir die passende Übersetzung, so wird diese sofort in die TeXShop Version auf meiner Webseite eingepflegt. Schreiben Sie einfach an koch@math.uoregon.edu.
