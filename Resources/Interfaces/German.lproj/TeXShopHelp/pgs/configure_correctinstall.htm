Seit Verion 1.34 beinhaltet TeXShop eine Funktion mit Namen pdfsync. Hiermit können Sie durch Klicken in das Vorschau-Fenster zum entsprechenden Punkt im Quelltext gelangen. Um diese Funktion zu benutzen, benötigen Sie die folgenden Dateien: "pdfsync.sty", "pdfsync.tex" und "pdfsync4context.tex". Diese müssen jeweils im Ordner „~/Library/texmf/tex/latex“, „~/Library/texmf/tex/plain“ und „~/Library/texmf/tex/context“ installiert sein. Löschen Sie eventuelle ältere Versionen von "pdfsync.sty" aus „~/Library/texmf/tex/latex“ falls vorhanden. Kopieren Sie dann die oben genannten Dateien aus der TeXShop Distribution, die Sie aus dem Internet geladen haben und kopieren Sie diese in die entsprechenden Ordner, wobei „~/Library“ der Library-Ordner Ihres Privat-Verzeichnisses ist. Sollten einige der oben genannten Ordner nicht existieren, müssen Sie diese erstellen.

Seit Version 1.35 werden mit TeXShop AppleScripts von Claus Gerhardt und Will Robertson mitgeliefert. Wenn Sie von einer älteren Verion aktualisieren erscheinen diese Skripte nicht automatisch im Menü „Makros“.

Wenn Sie die Standardmakros verändert haben, können Sie die Skripte erhalten, indem Sie den Ordner „Macros“ aus dem Ordner „~/Library/TeXShop“ an einen anderen Ort verschieben. Beim nächsten Start wird TeXShop den Ordner mit den neuen Skripten neu anlegen. Nachdem TeXShop den neuen Ordner angelegt hat, können Sie Ihre angepassten Makros mit Hilfe des Makro-Editors wieder einfügen.

TeXShop 1.35 wird mit neuen Vorlagen von Will Robertson ausgeliefert. Diese Vorlagen werden nicht automatisch installiert. Sie können Sie jedoch erhalten, indem Sie den vorhandenen Ordner „Templates“ im Ordner „~/Library/TeXShop“ auf den Schreibtisch verschieben. Beim nächsten Start wird der Vorlagenordner mit den neuen Vorlagen erstellt. Sie können jetzt ihre angepassten Vorlagen vom Schreibtisch wieder in den Ordner „~/Library/TeXShop/Templates“ zurück kopieren.
