Mit TeXShop 1.35 wird ein neuer Suchdialog eingeführt. Um dieses zu benutzen muss mindestens Mac OS X 10.3 (Panther) installiert sein. Anderen Benutzern wird noch der alte Dialog angezeigt. Sie können allerdigns auch bewusst zum alten Dialog zurückkehren, indem Sie dies in den TeXShop-Einstellungen angeben. Die meisten werden jedoch den neuen Dialog auf Grund der neuen Funktionen bevorzugen. Einer der Vorzüge ist, dass nun auch reguläre Ausdrücke unterstützt werden. Sollten Sie das Gefühl haben, dass die Suche nicht so verläuft wie Sie sich es wünschen, sollten Sie einige Ämderungen an den Suchparametern vornehmen. Diese Vorgaben werden natürlich auch gesichert.

Folgende Anmerkungen richten sich an Benutzer von TeXShop 1.33 oder früher. Im TeXShop Einstellungsdialog, sollten unter der Registerkarte „Programme“ folgende Änderungen vorgenommen werden:

  1. Die pdftex Einstellung sollte sein:
    pdftex --shell-escape
  2. Die pdflatex Einstellung sollte sein:
    pdflatex --shell-escape
  3. Die altpdftex Einstellung sollte sein:
    altpdftex --maxpfb
  4. Die altpdflatex Einstellung sollte sein:
    altpdflatex --maxpfb

Japanische Benutzer der Version 1.26 oder früher sollten die Datei „~/Library/TeXShop/LatexPanel/completion.plist“ durch die englische Version ersetzen, da die Modifizierung für das Japanische (ShiftJIS ¥) nicht länger nötig ist. Entfernen Sie hierfür die Datei completion.plist aus dem Ordner. Eine neue Datei wird beim nächsten Start von TeXShop angelegt.
