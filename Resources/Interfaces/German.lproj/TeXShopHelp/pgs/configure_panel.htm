Die LaTeX Palette enthält einen Titel mit dem Namen „Angepasst“. Bis zu 16 benutzerdefinierte Knöpfe können hier eingefügt werden. Andere Knöpfe der LaTeX Palette können jedoch nicht geändert werden. Jedoch kann auch hier der Text, der beim Drücken des Knopfes eingefügt wird, verändert werden.

Die Kommentare am Anfang der Datei „completions.plist“ erklären wie Veränderungen vorgenommen werden. Ändern Sie einfach die Datei „~/Library/TeXShop/Keyboard/completions.plist“ nachdem Sie sie mit TeXShop geöffnet haben. Stellen Sie sicher, dass Sie sie mit UTF-8 Kodierung sichern, wenn Sie Unicode Zeichen verwenden.
