TeXShop-64

TeXShop-64 is a new version of TeXShop for Lion (the current TeXShop 2.43 also works fine there). For a long time to come, TeXShop and TeXShop-64 will be developed in parallel. TeXShop's "Check for Updates..." item will never update TeXShop to TeXShop-64 or TeXShop-64 to TeXShop, but only update "within the TeXShop family," or "within the TeXShop-64 family." You can install both TeXShop and TeXShop-64 and switch back and forth between these programs.

MacTeX and TeXShop on Lion

The MacTeX Working Group from the Tex User Group (TUG) constructed an install package which installs everything needed to run TeX on Mac OS X in one step. This package is free, and uses Apple's standard installer; installation takes four to eight minutes and is automatic. The package installs TeX Live, the complete reference edition of TeX produced in cooperation by TeX User Groups across the world. It also installs several GUI utilities for TeX including TeXShop, so it is not necessary to get the front end separately. Everything is completely configured and ready to use once the installer finishes its job.

MacTeX-2011 was released in early July 2011, and has been extensively tested on Lion. To obtain it, click the following link: MacTeX.mpkg.zip.

Actually, MacTeX-2010, released a year ago, installs and works fine on Lion. But TeXShop from that release needs to be updated. All releases of TeXShop from version 2.39 on work on Lion; that includes the latest release, TeXShop 2.43.

The new version of TeXShop available above requires Lion and takes advantage of new interface features in the operating system. It is not in MacTeX-2011.

The ~/Library Folder in Lion

There is one small change in Lion which affects TeX. Each user's home directory contains a folder named ~/Library where configuration information for that user is stored. On Lion, this folder is hidden. But it is easy to open the folder. Select the Finder's "Go" menu, which has commands to open various important folders. When the "Option" key is pressed, "Library" is added to this list, and the new item takes you to your personal Library folder.

Recall that the ~/Library folder contains configuration files for TeXShop and TeXworks, and may contain a texmf folder for a user's personal style and font additions to TeX.
