MakeIndex es una herramienta destinada a añadir un índice alfabético a un documento LaTeX. Para crear tal índice, debes

  1. Añadir el comando «\usepackage{makeidx}» en la parte de arriba del documento con el código fuente LaTeX.
  2. Poner el comando «\makeindex» en el preámbulo del documento.
  3. Poner el comando «\printindex» donde deba aparecer el índice, generalmente justo delante de «\end{document}».
  4. Insertar las entradas para el índice en el documento con el código fuente, como en el siguiente ejemplo:
Hay muchos animales en el mundo\index{animal}. Por ejemplo, osos\index{animal!oso}
y tigres\index{animal!tigre}, así como varias clases de insectos\index{insecto|see{animal}}.

Para crear el índice alfabético, componer el documento de la manera usual. Entonces ejecutar MakeIndex. Finalmente, componer de nuevo.

Se pueden encontrar muchos detalles adicionales en The LaTeX Companion, de Goossens, Mittelbach y Samarin.
