Mathematica puede guardar imágenes en formato eps. Se pueden usar directamente sea cual sea el motor de composición usado (latex o pdflatex); tal como se explica en la correspondiente sección, estos archivos se convierten automáticamente a pdf si se usa pdflatex.

Los archivos eps generados por Mathematica usan las convenciones de Macintosh para los fines de línea. Versiones antiguas de TeXShop y teTeX requerían que convirtiéramos estos fines de línea a los usuales en Unix antes de usar esos archivos eps para convertirlos a formato pdf. Pero esto ya no es necesario.
