El grupo de trabajo MacTeX del TeX Users Group (TUG) ha confeccionado un paquete instalable (del tipo de los que proporciona Apple) conteniendo todo lo necesario para utilizar TeX en Mac OS X. El paquete se instala en un par de minutos, y casi no necesita intervención del usuario. Contiene una muy completa versión de la redistribución de TeX de Gerben Wierda, así como TeXShop, BibDesk, el verificador ortográfico Excalibur, y el i-Installer de Gerven. Para acceder a un link actualizado a este paquete, ve a

http://pages.uoregon.edu/koch/texshop/obtaining.html

También puedes conseguir por separado la redistribución de Gerben Wierda (ésta la distribución contenida en el paquete antes citado). La última versión de esta distribución está compilada tanto para microprocesadores Intel como PowerPC, y requiere Mac OS X 10.3 o superior. Está basada en el último teTeX para Mac OS X, y también incluye programas (actualizados) que acompañan a TeXLive, la distribución del TeX User's Group. Para conseguir estos archivos y programas, basta ir a la página web de Wierda

http://www.rna.nl/tex.html

y descagar i-Installer, el instalador de TeX. En particular, el instalador está incluido en el paquete II2.dmg. Ejecuta el instalador e instala los siguientes paquetes en el orden en el que aparecen:

  • FreeType 2
  • libwmf
  • PNG Library
  • Ghostscript 8
  • ImageMagick
  • FontForge
  • Fondu Mac Font cli Tools
  • TeX

El paquete MacTeX instala algunos otros paquetes adicionales, que también se pueden conseguir con i-Installer:

  • CM-Super
  • CB Greek
  • MusixTeX
  • ConTeXt updated
  • XeTeX (éste está en el directorio de Jonathan Kew)

Se pueden encontrar instrucciones de instalación más detalladas en la página web de Wierda, y también en la de TeXShop. Ambas aparecen más adelante en esta ayuda.
