Los programas desarrollados con Cocoa heredan automáticamente la tecnología de corrección ortográfica de Apple. El verificador ortográfico incorporado en Mac OS X 10.4 y superior se comporta mejor que los de las versiones anteriores cuando se enfrenta a archivos en LaTeX. Pero no entiende los comandos de LaTeX, y marca muchos de ellos como palabras mal escritas. Afortunadamente, se pueden añadir otros servicios a la tecnología de verificación ortográfica original de Apple; y estarán disponibles para todos los programas Cocoa.

TeXShop soporta verificación ortográfica mientras se escribe, que se activa o desactiva con un comando del menú. En el diálogo de Preferencias hay un ítem destinado a establecer cuál tiene que ser su estado cada vez que se arranca TeXShop.

Aspell es «a more intelligent Ispell» (‘un Ispell más inteligente’) para máquinas Unix escrito por Kevin Adkinson. Este programa ha sido adaptado a Mac OS X y convertido en un servicio de verificación ortográfica por Anton Leuski. Además, soporta LaTeX. Para obtenerlo, ir a

http://cocoaspell.leuski.net/

Otro corrector ortográfico para LaTeX es Excalibur. Este programa, escrito por Rick Zaccone, puede abrir archivos de TeXShop. Para obtenerlo, ir a

http://www.eg.bucknell.edu/~excalibr/excalibur.html
