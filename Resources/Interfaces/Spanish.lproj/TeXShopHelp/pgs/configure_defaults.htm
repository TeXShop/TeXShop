TeXShop 1.35 viene con un nuevo panel de Búsquedas, que requiere Mac OS X 10.3. Los demás usuarios continuarán viendo el antiguo panel. De todas formas, seleccionando el correspondiente ítem en las Preferencias de TeXShop, se puede seguir usando el antiguo panel, pero la mayoría de los usuarios encontrarán que el nuevo es mucho más potente; soporta expresiones regulares. Al principio, puede que este panel no se comporte como esperas. Prueba con cuidado. seleccionando entre las diversas opciones que incorpora, hasta que lo logres. Las sucesivas veces que lo uses, el panel recuerda cómo lo habías configurado.

Los restantes comentarios son para los que ya eran usuarios de versiones anteriores a la 1.33, y que pueden no haber hecho estos cambios antes. En el panel de las preferencias de TeXShop, y tras elegir la solapa Motor, se deben hacer los siguientes cambios:

  1. La preferencia para pdftex debe ser
    pdftex --shell-escape
  2. La preferencia para pdflatex debe ser
    pdflatex --shell-escape
  3. La preferencia para altpdftex debe ser
    altpdftex --maxpfb
  4. La preferencia para altpdflatex debe ser
    altpdflatex --maxpfb

Los usuarios japoneses de TeXShop 1.26 o anterior deben reemplazar el archivo ~/Library/TeXShop/LatexPanel/completion.plist con la versión inglesa, puesto que ya no es necesaria ninguna modificación especial para Japanese(ShiftJIS ¥). Para que esto ocurra, borra el archivo completion.plist de su carpeta; el nuevo archivo se creará la siguiente vez que arranques TexShop.
