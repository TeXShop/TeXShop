En el menú Ventana podemos encontrar el ítem «Panel matrices...». Eligiéndolo, se abre un panel que facilita la creación de matrices. El espacio para cada entrada de la matriz es muy pequeño; si queremos escribir algo más grande, una buena idea es editarlo en otro sitio, copiarlo, y pegarlo en la matriz. Este magnífico panel ha sido fruto del trabajo de Jonas Zimmermann, cuyo correo electrónico es zimmerleut@gmx.de.

En TeXShop 1.35, «Panel matrices...» también puede crear tablas. Este panel se puede configurar usando el archivo «matrixpanel_1.plist» de ~/Library/TeXShop/MatrixPanel. La estructura de este archivo ha sido revisado en TeXShop 1.35; posiblemente también tengas un archivo antiguo llamado «matrixpanel.plist», que provendrá de TeXShop 1.34.

Hay una preferencia oculta para asignar un tamaño predeterminado a las matrices que crea el panel. Para cambiar el valor que viene por defecto, usa

defaults write TeXShop matrixsize 12
