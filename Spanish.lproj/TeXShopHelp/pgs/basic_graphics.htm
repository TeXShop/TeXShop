Los programas pdftex y pdflatex pueden usar archivos gráficos (previamente producidos por otros programas) que tengan formato pdf, jpg, o tif. Un peculiaridad es que los archivos tiff deben tener extensión «.tif» en lugar de «.tiff». El formato nativo de gráficos de Mac OS X es pdf (portable document format), y tales archivos se imprimen bien en cualquier tamaño. Es previsible que la mayoría de los futuros programas gráficos para Mac generen archivos pdf.

Si usabas TeX en el pasado, tus ilustraciones estarían, posiblemente, en formato eps. Pero pdftex y pdflatex no pueden usar tales archivos directamente, sino que las ilustraciones deben ser convertidas a formato pdf. Para convertir un archivo eps con una ilustración, abrirlo con TeXShop. La ilustración aparecerá en una ventana gráfica y, simultáneamente, TeXShop guardará el correspondiente archivo pdf en el disco. También Ghostscript contiene un programa (ejecutable desde la línea de comandos) para efectuar tales conversiones; de hecho, TeXShop llama a este programa. Para convertir myfile.eps a myfile.pdf dentro del Terminal, teclea

epstopdf myfile.eps

Los autores de pdflatex y los autores de paquete gráfico graphicx han previsto una manera fácil de incluir archivos en un documento LaTeX compuesto con pdflatex, incluso si el documento va a ser compuesto más tarde con latex estándar para generar un dvi que se podrá distribuir a otras personas. Para ello, al comienzo del fichero con código LaTeX, incluye la línea

\usepackage{graphicx}

Cuando desees insertar un archivo gráfico, digamos «f1.pdf», usa el comando

\includegraphics[width=2in]{f1}

Este comando hará que tex use el archivo «f1.pdf» cuando el código se compone con pdflatex, pero use el archivo «f1.eps» cuando el código se compone con latex.

Si te has actualizado desde una versión previa de TeXShop, y quieres usar la conversión automática de archivos eps y tif al componer, asegúrate de que

  1. Has instalado Ghostscript 8, Freetype 2, wmf and iconv conversion support, y ImageMagick con el instalador de Gerben Wierda.
  2. La preferencia para el programa pdflatex es «pdflatex --shell-escape» (solapa Motor del menú Preferencias).
  3. La cabecera del fichero LaTeX contiene las siguientes líneas:
\usepackage{graphicx}
\usepackage{epstopdf}
\DeclareGraphicsRule{.tif}{png}{.png}{`convert #1 `basename #1 .tif`.png}
