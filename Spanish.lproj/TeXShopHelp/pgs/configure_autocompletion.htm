Autocompleción se activa y desactiva en la solapa Documento de las Preferencias de TeXShop; por defecto, aparece desactivado. Cuando está activada, al teclear ciertos caracteres se insertan otros en el archivo con el código fuente. Por ejemplo, al teclear ^ aparece ^{ }, y el cursor se coloca entre las llaves.

La autocompleción se puede cambiar y añadir más cosas; para hacerlo hay que editar el archivo «autocompletion.plist». Los comentarios al comienzo del archivo explican cómo realizar tales cambios. Para ello, edita el archivo ~/Library/TeXShop/Keyboard/autocompletion.plist con TeXShop. Asegúrate de editar y guardar el archivo con formato UTF-8 si usas caracteres Unicode.
