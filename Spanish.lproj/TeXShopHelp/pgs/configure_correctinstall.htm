¿Has instalado TeXShop correctamente?

Desde la versión 1.34, TeXShop dispone de una utilidad denominada pdfsync: al hacer clic en algún punto de la ventana con el pdf previsto, se abre la ventana correspondiente al código TeX y, en ella, aparece seleccionada la línea de código apropiada. Para que esto funcione, es necesario que teTeX tenga instalado un paquete llamado «pdfsync.sty». Busca este archivo en la distribución de TeXShop y arrástralo a ~/Library/texmf/tex/latex. Aquí, ~/Library se refiere a la carpeta Library de tu directorio de usuario (‘home’). Si no lo has hecho antes, puede que necesites crear alguna de las carpetas texmf, tex, latex.

Desde la versión 1.33, TeXShop viene con applescripts adicionales realizados por Claus Gerhardt. Si estás actualizando desde una versión anterior, no los verás automáticamente.

Si no has editado nunca el conjunto de macros predeterminadas, puedes activar las nuevos macros yendo a la carpeta ~/Library/TeXShop y borrando toda la carpeta Macros que hay allí. La siguiente vez que lo arranques, TeXShop volverá a crear esta carpeta con las últimos macros. No es suficiente borrar el contenido de la carpeta Macros; hay que borrar también la carpeta misma.

Si has editado las macros que venían por defecto, y añadido algunas nuevas, borra temporalmente la carpeta Macros y deja que TeXShop cree una nueva. Entonces, añade tus macros nuevas con el editor de macros.
