Preferencias predeterminadas

En el panel de las preferencias de TeXShop, y tras elegir la solapa Motor, se deben hacer los siguientes cambios:

  1. La preferencia para pdftex debe ser
    pdftex --shell-escape
  2. La preferencia para pdflatex debe ser
    pdflatex --shell-escape
  3. La preferencia para altpdftex debe ser
    altpdftex --maxpfb
  4. La preferencia para altpdflatex debe ser
    altpdflatex --maxpfb

Los usuarios japoneses de TeXShop 1.26 o anterior deben reemplazar el archivo ~/Library/TeXShop/LatexPanel/completion.plist con la versión inglesa, puesto que ya no es necesaria ninguna modificación especial para Japanese(ShiftJIS ¥). Para que esto ocurra, borra el archivo completion.plist de su carpeta; el nuevo archivo se creará la siguiente vez que arranques TexShop.
