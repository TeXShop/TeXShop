El panel LaTeX contiene una solapa llamada «Personal»; el usuario puede añadir en ella hasta dieciséis botones. El resto de los botones del panel LaTeX no se pueden modificar, pero sí que se puede cambiar el texto que se inserta al pulsar el cada botón.

Los comentarios al comienzo del archivo «completions.plist» explican cómo realizar estos cambios. Para ello, edita el archivo ~/Library/TeXShop/LatexPanel/completions.plist con TeXShop. Asegúrate de editar y guardar el archivo con formato UTF-8 si usas caracteres Unicode.
