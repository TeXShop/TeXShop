Métodos alternativos de composición

Hay otra manera de componer (‘typeset’) con TeXShop; con este método alternativo, las ilustraciones eps se pueden incluir directamente sin convertirlas previamente a pdf. Para ello, elegir «TeX y Ghostscript» en el menu Componer. Entonces, TeXShop llamará a tex o latex (en lugar de a pdftex o pdflatex) y generará un archivo «.dvi» (en lugar de un «.pdf»), luego llamará a dvips para convertirlo en un archivo postscript, y finalmente llamará a ps2pdf para convertir el archivo postscript a pdf. El método elegido en el menú Componer sólo afectará al archivo tex que se encuentre en ese momento al frente; para otros documentos, TeXShop continuará usando pdftex o pdflatex. Si deseamos cambiar el método predeterminado para componer, podemos hacerlo mediante del diálogo de preferencias.

El método «TeX y Ghostscript» puede ser preferible para proyectos antiguos con muchas ilustraciones en formato eps, para archivos tex con comandos postscript especiales, y para archivos tex que incluyan letras bitmap que no se verán correctamente cuando se compongan mediante pdflatex. La experimentación determinará qué «motor de composición» nos conviene usar.

Hay una manera para establecer de forma permanente en un archivo el método de composición, independientemene de las preferencias establecidas. Si la primera línea del documento es %&tex, sin ningún espacio inicial en la línea, entonces se usará el método tex + ghostscript. Si la primera línea es %&latex, entonces se usará latex + ghostscript. Y si la primera línea es %&pdftex o %&pdflatex, entonces se usará pdftex o pdflatex. Finalmente, si la primera línea es %&personaltex o %&personallatex, entonces se usará el correspondiente script personal señalado en el diálogo de preferencias.
