Texto coloreado en TeX

Siempre ha sido posible colorear la salida que proporciona LaTeX. Para hacerlo, añade la línea

\usepackage{color}

en el preámbulo, e inserta una línea como

\color{blue}

antes de que LaTeX comience a componer páginas. Para colorear una sección limitada de texto, usa un comando como

{\color{red} !`Esto es muy importante!}

Para definir colores a medida, usa un comando como

\definecolor{mycolor}{rgb}{0.2, 0.7, 0.8}

Cuando coloreamos la salida, puede ser difícil leerla en la ventana con el pdf previsto. Por otra parte, TeXShop tiene preferencias ocultas para ponerle color de fondo a esa ventana. Este color de fondo no aparecerá cuando imprimamos el documento. Para establecer color de fondo gris, teclea los siguientes comandos en la Terminal:

defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_R 0.5
defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_G 0.5
defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_B 0.5

Cambia 0.5 por 1 para obtener blanco y negro. Si vas a cambiar el color de fondo muy a menudo, puedes crear, con el editor de macros, una macro AppleScript llamada «PDF con fondo gris» que contenga el siguiente texto:

--AppleScript
do shell script "defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_R 0.5"
do shell script "defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_G 0.5"
do shell script "defaults write edu.uoregon.math.koch.TeXShop Pdfbackground_B 0.5"

y crea una macro similar llamada «PDF con fondo blanco».
