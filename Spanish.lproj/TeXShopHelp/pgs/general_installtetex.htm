Cómo obtener e instalar teTeX

Gerben Wierda ha compilado el último teTeX para Mac OS X. Su versión también incluye los programas (actualizados) que acompañan a TeXLive, la distribución del TeX User's Group. Para conseguir estos archivos y programas, ir a la página web de Wierda

http://www.rna.nl/tex.html

y descagar i-Installer, el instalador de TeX. En particular, el instalador está incluido en el paquete II2.dmg. Al ejecutar el instalador, instalar los siguientes paquetes:

  • TeX
  • Ghostscript 8
  • Freetype 2
  • wmf and iconv conversion support
  • ImageMagick
  • CM Super for TeX

El primer paquete es esencial para que funcione TeX. El segundo es necesario para usar la opción TeX + Ghostscript de TeXShop, descrita más adelante. Los paquetes tercero, cuarto y quinto sirven para convertir diversos formatos gráficos en otros que entienda TeX. El paquete final, que es opcional, contiene un conjunto de tipos de letra adicionales.

Si estás usando Mac OS X 10.3 (Panther) o posterior, puedes no necesitar instalar Ghostscript. Panther incluye su propio ‘distiller’ para convertir archivos postscript a formato pdf, y TeXShop tiene una preferencia para indicarle que use Distiller en lugar de Ghostscript. Pero no está de más instalarlo, pues algunos paquetes llaman directamente a Ghostscript. El más importante es epstopdf.sty, que sirve para convertir automáticamente archivos eps en pdf al componer.

Se pueden encontrar instrucciones de instalación más detalladas en la página web de Wierda, así como en la de TeXShop.
