Preliminares

Las macros de TeXShop y el editor de macros han sido implementadas por Mitsuhiro Shishikura. Las macros predefinidas han sido creadas por Mitsuhiro Shishikura y Hirokazu Ogawa.

Las macros son de dos clases. Algunas insertan texto en el código fuente TeX; estas macros son similares a los botones del panel LaTeX. Otras macros ejecutan scripts programados en AppleScript.

Las macros se almacenan en ~/Library/TeXShop/Macros/Macros.plist. Si este archivo no aparece al ejecutar TeXShop, se crea un archivo con las macros predefinidas. El archivo Macros.plist es un archivo de texto ordinario, así que se puede abrir e inspeccionar con TeXShop. Pero no es necesario, pues es mejor manipular este fichero haciendo uso del editor de macros.

Cuando defines macros útiles que desear compartir con otros usuarios, abre el editor de magros y elige «Guardar selección en archivo...» en el menú Macros. Esto creará un archivo en el lugar que desees. Para distribuir tus macros, envía este archivo.

Para añadir macros creadas por otros a tu lista de macros, abre el editor de macros y elige «Añadir macros desde archivo...» en el menú Macros. Entonces usa el editor de macros para organizarlas como desees.
