Las macros de TeXShop y el editor de macros han sido implementadas por Mitsuhiro Shishikura. Las macros predefinidas para LaTeX han sido creadas por Mitsuhiro Shishikura y Hirokazu Ogawa. Las macros predefinidas para ConTeXt han sido creadas por Hans Hagen.

Las macros disponibles dependen del motor de composición que se esté usando. Si es LaTeX, aparecen una serie de macros adecuadas para LaTeX. Si es ConTeXt, las de ConTeXt. Al instalar TeXShop por primera vez, sólo dispone de macros para estos dos motores; para cualquier motor que no sea ConTeXt, las macros que se muestran son las de LaTeX.

Las macros son de dos clases. Algunas insertan texto en el código fuente TeX; estas macros son similares a los botones del panel LaTeX. Otras macros ejecutan scripts programados en AppleScript.

Las macros se almacenan en ~/Library/TeXShop/Macros/Macros_Latex.plist y ~/Library/TeXShop/Macros/Macros_Context.plist. Si alguno de estos archivos no aparece al ejecutar TeXShop, se crean automáticamente, con las macros predefinidas. Los archivos Macros_Latex.plist y Macros_Context.plist son archivos de texto ordinario, así que se pueden abrir e inspeccionar con TeXShop. Pero no es necesario, pues es mejor manipularlos por medio del editor de macros.

El editor de macros modifica las mcaros almacenadas en Macros_Latex.plist o Macros_Context.plist según cual sea el motor de composición que hayamos elegido, LaTeX o ConTeXt, respectivamente. Pero si hemos elegido otro motor de composición, como Plain TeX, por ejemplo, el editor de macros reemplaza el conjunto de macros predeterminadas de LaTeX por el nuevo conjunto definido por el usuario para Plain TeX, y lo almacena en Macros_Tex.plist; obviamente, deja Macros_Latex.plist sin cambiar.

Cuando defines macros útiles que desear compartir con otros usuarios, abre el editor de magros y elige «Guardar selección en archivo...» en el menú Macros. Esto creará un archivo en el lugar que desees. Para distribuir tus macros, envía este archivo.

Para añadir macros creadas por otros a tu lista de macros, abre el editor de macros y elige «Añadir macros desde archivo...» en el menú Macros. Entonces usa el editor de macros para organizarlas como desees.
